隐私权政策

2026 年 6 月 14 日

数据收集

Torn Company Manager 不收集、不传输、不存储任何用户个人数据。本扩展的所有数据均保存在用户本地浏览器的存储空间中（IndexedDB 与 chrome.storage.local），不会发送到任何第三方服务器。

网络请求

本扩展仅向以下 Torn 游戏官方及社区 API 发起请求，用于获取游戏内公司数据：

  • api.torn.com — Torn 官方 API
  • yata.yt — YATA 社区 API

这些请求仅用于读取公开的游戏数据，不涉及用户身份信息。

数据存储

所有数据（员工信息、训练记录、库存、财务、设置等）仅存储在用户浏览器本地。用户可随时通过扩展设置页面清除所有数据。

数据共享

本扩展不会将任何数据出售、共享或传输给任何第三方。

儿童隐私

本扩展不面向 13 岁以下儿童，也不收集任何儿童的个人信息。

政策更新

如有政策变更，将在本页面更新日期。建议用户定期查阅。

联系方式

如有疑问，请通过 Chrome Web Store 的开发者联系方式与我们沟通。
